# リリースノート v1.0.0

リリース日: 2026年3月

---

## 概要

AccessRoute v1.0.0 は、バリアフリールートナビゲーションアプリの初回リリースである。車椅子、ベビーカー、杖歩行、高齢者など、さまざまな身体状況に合わせたバリアフリールートとスポット情報を提供する。

---

## 搭載機能

### プロファイル設定

- 5ステップのオンボーディングフローで移動手段・同行者・最大移動距離・回避条件・希望条件を登録
- プロファイルはいつでも変更可能

### バリアフリールート検索

- 出発地と目的地を指定し、プロファイルに基づくバリアフリールートを複数候補で提案
- 3つの優先モード: 最短ルート、安全優先ルート、バリアフリー優先ルート
- 各ルートにアクセシビリティスコア（0〜100）を表示
- 区間ごとの詳細情報: 方向案内、距離、所要時間、階段の有無、坂道の勾配、路面の種類
- 注意が必要な区間への警告表示（階段あり、急な坂道等）

### 沿道スポット情報

- ルート沿いのバリアフリー施設を自動検索・表示
- 10種類のカテゴリ: バリアフリートイレ、休憩所、レストラン、カフェ、公園、キッズスペース、授乳室、エレベーター、駐車場、その他
- 各スポットのバリアフリー詳細情報: 車椅子対応、エレベーター有無、おむつ交換台、授乳室、床の種類

### 周辺スポット検索

- 現在地や指定地点の周辺にあるバリアフリースポットを検索
- カテゴリや検索範囲の指定が可能

### AIチャット機能

- テキスト対話形式で移動のニーズを伝達
- AIがトラベルコンシェルジュとして応答し、ルートやスポットを提案
- 会話からユーザーのニーズ（移動手段、回避条件等）を自動抽出

### アクセシビリティ対応

- 全画面で VoiceOver によるスクリーンリーダー操作に対応
- Dynamic Type によるフォントサイズ変更に対応
- 高コントラストモード対応
- 最小タップターゲット 44x44pt 以上を確保
- カラーブラインド対応（色以外の視覚的手がかりを併用）

---

## 動作環境

- iOS 17.0 以上
- iPhone / iPad 対応
- 日本語

---

## 既知の制限事項

- ルートのバリアフリー情報は Google Maps Platform のデータに基づく推定であり、実際の現地状況と異なる場合がある
- 工事や通行止め等のリアルタイム情報は反映されない
- AI応答は言語モデルによる生成テキストであり、内容の正確性を完全に保証するものではない
- 対応言語は現時点では日本語のみ
