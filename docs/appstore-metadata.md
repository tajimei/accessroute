## 説明文（日本語、4000文字以内）

AccessRoute は、車椅子やベビーカー、杖歩行、高齢者の方など、さまざまな身体状況に合わせたバリアフリールートを提案するナビゲーションアプリです。

お出かけ前の不安を解消し、誰もが安心して外出できる社会を目指して開発しました。

---

【こんな方におすすめ】

- 車椅子で移動される方
- ベビーカーでお出かけされる方
- 杖を使って歩行される方
- ご高齢の方やそのご家族
- 障がいのある方と一緒にお出かけされる方
- バリアフリーな施設を探している方

---

【主な機能】

■ パーソナライズされたルート検索

初回起動時に移動手段（車椅子、ベビーカー、杖、徒歩など）、同行者、最大移動距離、回避したい条件（階段、急な坂道、混雑、暗い道）、希望する施設（トイレ、休憩所、屋根付き経路）を登録します。以降のルート検索では、これらの情報に基づいて最適なバリアフリールートを自動的に提案します。

■ アクセシビリティスコア

ルートやスポットごとに 0 から 100 のスコアでバリアフリー度を数値化。一目でルートの安全性やスポットの利用しやすさを判断できます。

■ 3つの検索モード

目的に合わせて「最短ルート」「安全優先ルート」「バリアフリー優先ルート」の3つから選択できます。それぞれの条件で複数のルート候補を提示し、比較しながら最適なルートを選べます。

■ 区間ごとの詳細情報

各ルートは区間（ステップ）ごとに分割され、方向案内、距離、所要時間に加え、階段の有無、坂道の勾配、路面の種類（舗装、砂利、土など）といったバリアフリー情報を表示します。注意が必要な区間には警告メッセージが表示されるため、事前に危険箇所を把握できます。

■ 沿道スポット情報

ルート沿いにあるバリアフリー対応施設を自動的に検索・表示します。バリアフリートイレ、休憩所、エレベーター、授乳室、キッズスペース、駐車場など、10種類のカテゴリでスポットを探せます。各スポットには車椅子対応の有無、エレベーターの有無、おむつ交換台の有無などの詳細なバリアフリー情報が掲載されています。

■ 周辺スポット検索

現在地や指定した地点の周辺にあるバリアフリースポットを検索できます。検索範囲やカテゴリを指定して、必要な施設をすぐに見つけられます。

■ VoiceOver 完全対応

すべての画面で VoiceOver によるスクリーンリーダー操作に対応しています。視覚に障がいのある方でも快適にご利用いただけます。画面内のすべてのボタン、カード、入力フィールドに適切なアクセシビリティラベルを付与し、44x44pt 以上のタップターゲットを確保しています。

■ プロファイルの柔軟な変更

登録したプロファイルはいつでも変更可能です。移動手段が変わった場合や、同行者が変わった場合も、すぐにルート検索の条件を更新できます。

## プライバシーポリシーの概要

### 収集する情報

| データの種類 | 収集目的 | 保存場所 |
|------------|---------|---------|
| 位置情報 | ルート検索の起点・現在地表示 | デバイスローカル / Firebase |
| ユーザープロファイル（移動手段、回避条件等） | ルート検索条件のパーソナライズ | Firebase Firestore |
| 認証情報（メールアドレスまたは外部認証） | ユーザー識別・データ保護 | Firebase Auth |
| チャット履歴（Phase 2） | AIによるニーズ抽出 | Firebase Firestore |

### データの取り扱い方針

- 収集したデータはアプリの機能提供のみに使用し、広告やマーケティング目的には使用しない
- ユーザーの位置情報は、ルート検索およびスポット検索の実行時にのみ送信される
- プロファイル情報は Firebase のセキュリティルールにより、本人のみがアクセスできるよう保護される
- データの削除はアプリ内から随時可能とする
- 第三者への個人情報の提供は行わない（ただし Google Maps API へのルート検索リクエストにおいて位置座標が送信される）

### 準拠法・規制

- 個人情報の保護に関する法律（個人情報保護法）に準拠する
- Apple の App Store Review ガイドラインに準拠する
- 位置情報の使用について、初回利用時にユーザーの明示的な同意を取得する
