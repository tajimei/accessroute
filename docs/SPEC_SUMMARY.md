# AccessRoute - 仕様書（概要版）

## 1. プロダクト概要

**AccessRoute** は、ユーザーの身体状況（車椅子、ベビーカー、杖、高齢者同行等）に合わせたバリアフリールート・スポット提案を行うモバイルアプリ。

- **対象**: 移動に制約のあるユーザーとその同行者
- **プラットフォーム**: iOS / Android（Expo + React Native）
- **対応地域**: 全国（北海道〜九州 + 新幹線、10地域対応）

---

## 2. 主要機能

| 機能 | 概要 |
|------|------|
| **ルート検索** | 出発地〜目的地のバリアフリールートを検索。階段・坂道・未舗装路を考慮したスコアリング |
| **乗換案内** | 独自の全国鉄道DB + Dijkstraアルゴリズムによる乗換ルート提案 |
| **マップ表示** | Google Maps上にルート・乗換駅・スポットを表示。路線カラー対応 |
| **スポット検索** | 周辺のバリアフリー対応施設（トイレ、休憩所、エレベーター等）を検索 |
| **AIチャット** | 日本語LLMによる対話でユーザーのニーズを自動抽出し、プロファイルに反映 |
| **プロファイル** | 移動方法・同行者・回避条件・希望条件を保存し、検索に反映 |

## 5. スコアリング

- **基本スコア**: 100点から階段(-30)、急坂(-25)、中坂(-15)、未舗装(-20)、乗換(-10〜15)を減算
- **重み付きスコア**: 同行者タイプ別に係数を適用（障がい者: 階段2.0倍、高齢者: 坂1.5倍等）
- **スポットスコア**: バリアフリー設備（エレベーター、車椅子対応等）に基づく加減算
