## リリース手順

1. `develop` -> `main` のPRをマージ
2. `main` ブランチでタグを作成: `git tag v1.0.0 && git push origin v1.0.0`
3. GitHub Actions のリリースワークフローが自動実行される
4. TestFlight ビルドの配布を確認
5. 内部テスターで最終確認
6. App Store Connect で審査提出
